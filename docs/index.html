What is FSE Tools?

Extra tools for finding a better flight, inside the site you already use.

FSE Tools is not meant to overhaul FSEconomy or replace its normal workflow. It adds focused tools on top of the existing pages so it is easier to evaluate assignments and plan a more suitable flight.

The airport dispatcher view is a good example. It brings the available jobs and aircraft into a more useful layout, helping you scan options faster while still working within the original FSEconomy page.

Install

Install in three steps.

  1. Install Tampermonkey. It is one of the most popular browser extensions across all major browsers and allows to customize and enhance the functionality of web pages. It's available via the official Tampermonkey website for Chrome, Microsoft Edge, Safari, Opera Next, and Firefox.
  2. Install the latest userscript. Tampermonkey works using userscripts. Open the FSE Tools release and confirm the prompt.
  3. Open FSEconomy. Visit https://server.fseconomy.net/, sign in, and confirm FSE Tools is visible.
